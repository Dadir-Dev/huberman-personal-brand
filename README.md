# 📚 Huberman Lab Podcast – Modern UI Clone

A fully responsive, visually polished **Huberman Lab Podcast website clone**, built using **HTML5**, **Tailwind CSS**, and modern UI/UX principles.  
This project recreates the look and feel of Andrew Huberman’s platform with smooth animations, interactive elements, and a structured layout.

## 🚀 Features

### 🎨 Modern UI & Aesthetics

- Sleek dark theme with professional styling
- Tailwind-powered transitions, glows, shadows, and hover effects
- Clean spacing, typography, and color balance

### 📱 Fully Responsive Layout

- Mobile-first design
- Collapsible mobile navigation menu
- Responsive grid for episodes and protocols
- Flexible images and containers

### 🧭 Smart Navigation

- Sticky header
- Desktop + mobile navigation
- Search bar
- CTA buttons (Explore Episodes / Watch on YouTube)

### 🎬 Episodes Section

- Modern episode cards featuring:
  - Episode thumbnails
  - Duration tag
  - Episode number
  - Episode summary
  - Listen Now button
- Hover-based glow animations

### 🧪 Protocols Section

- Four science-based protocol cards
- Emoji icons + bold colors
- Smooth scale + hover transitions

### 🧠 About Section

- Clean two-column layout
- Huberman biography text
- Credentials tiles (Professor, Neuroscientist, Podcast Host, etc.)
- Research focus cards with icons

### 📰 Footer

- Quick navigation links
- Platforms (YouTube, Spotify, Apple Podcasts, Google Podcasts)
- Newsletter subscription UI
- Smooth back-to-top button

## 🛠️ Tech Stack

| Technology            | Purpose                          |

## 📁 Project Structure

```
/ (root)
├── src/     # Source files
│   ├── index.html  # Main HTML file
│   ├── images/     # Image assets
│   └── styles/     # Tailwind styles
├── dist/    # Output files
│   └── output.css  # Compiled Tailwind styles
├── postcss.config.js  # PostCSS configuration
├── tailwind.config.js # Tailwind configuration
└── package.json      # Project metadata & dependencies
```
